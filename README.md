# Integração Slack + Cursor

Integra o Cursor AI Cloud Agent ao Slack com acesso ao GitHub, permitindo que equipes solicitem tarefas de codificação, revisão e criação de pull requests diretamente de conversas no Slack, sem necessidade de abrir o editor.

Stack: Cursor Cloud Agent, Slack, GitHub
